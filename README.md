# My Portfolio (Full Advanced)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview production build:
   ```bash
   npm run preview
   ```

## Notes
- Replace `Your Name` and content in components.
- Add images to `src/assets` and update `src/data/projects.js`.
- Hook contact form to a backend or email service (Netlify Forms, Formspree).
